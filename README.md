# 恋爱脑自测器

一个纯静态的中文趣味测试网页，用于测试“想谈恋爱”的主要动机维度。

## 本地预览

直接用浏览器打开 `index.html` 即可。

```text
index.html
```

不需要安装依赖，也不需要启动服务器。

## 文件结构

```text
恋爱脑自测器/
├── index.html   页面结构
├── styles.css   页面样式
├── app.js       题库、计分、结果页、保存图片逻辑
└── README.md
```

## 修改内容

- 题库：在 `app.js` 顶部的 `questions` 数组里修改。
- 结果文案：在 `app.js` 的 `results` 对象里修改。
- 正式网址/二维码链接：修改 `app.js` 里的 `RESULT_LINK_OVERRIDE`。

如果 `RESULT_LINK_OVERRIDE` 为空，页面和二维码会默认使用当前打开网页的地址。

## 计分规则

每题 0-6 分：

- 非常赞同：6
- 赞同：5
- 有点赞同：4
- 中立：3
- 有点反对：2
- 反对：1
- 非常反对：0

每个维度取该维度题目的平均分，并按平均分从高到低排序。

当所有维度平均分都低于 3 分时，显示特殊结果“没那么想谈恋爱”；否则显示最高维度和第二高维度对应的话语。
